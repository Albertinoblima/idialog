Google Analytics 4

Como configurar:
  1. Crie uma propriedade GA4 e copie o Measurement ID (G-XXXXXXXX)
  2. No Google Cloud, habilite a Google Analytics Data API (gratuita)
  3. Crie uma conta de serviço e baixe o JSON da chave
  4. Copie o Property ID (número) do painel GA4
  5. Adicione no Railway: GA4_PROPERTY_ID e GA4_SERVICE_ACCOUNT_JSON
Cole o conteúdo completo do arquivo JSON. Armazenado de forma segura (encriptado).
