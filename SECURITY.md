# Política de Segurança - iDialog

## 🔒 Versões Suportadas

| Versão | Suportada          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## 🛡️ Reportar Vulnerabilidades

Se você descobrir uma vulnerabilidade de segurança, por favor:

1. **NÃO** abra uma issue pública
2. Envie um email para: security@idialog.com.br
3. Inclua o máximo de detalhes possível:
   - Descrição da vulnerabilidade
   - Passos para reproduzir
   - Impacto potencial
   - Versão afetada

## ⏱️ Tempo de Resposta

- **Primeira resposta**: 24-48 horas
- **Análise completa**: 5-7 dias úteis
- **Correção**: Dependendo da severidade

## 🏆 Reconhecimento

Reconhecemos publicamente pesquisadores responsáveis que reportam vulnerabilidades, a menos que solicitem anonimato.

## 📋 Diretrizes de Divulgação Responsável

- Aguarde nossa confirmação antes de divulgar publicamente
- Dê tempo razoável para correção (90 dias)
- Evite causar danos aos usuários ou sistemas

## 🛠️ Medidas de Segurança Implementadas

- Validação de entrada em formulários
- Sanitização de dados
- Headers de segurança HTTP
- HTTPS obrigatório em produção
- Content Security Policy (CSP)

---

**Segurança é prioridade - obrigado por ajudar! 🔐**
