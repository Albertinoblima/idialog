Informações Técnicas do Site iDialog

Estrutura

  • Site 100% estático: HTML, CSS, JavaScript.
  • Organização em pastas: pages/, components/, assets/, public/, src/.
  • Componentes de cabeçalho e rodapé dinâmicos via JavaScript (components/header.html, components/footer.html).
  • Estilos centralizados em src/styles/main.css e src/styles/responsive.css.
  • Scripts principais em assets/js/main.js e assets/js/components.js.

Acessibilidade

  • Imagens com alt descritivo e loading="lazy".
  • Hierarquia de títulos e navegação por teclado (tabindex em títulos principais).
  • Contraste de cores e botões acessíveis.
  • Links institucionais claros e destacados.

Responsividade

  • Layout responsivo para desktop, tablet e mobile.
  • Grid flexível e espaçamento consistente.

Interatividade

  • Imagens de portfólio e ícones são clicáveis e levam para páginas ou arquivos relevantes.
  • Menu dinâmico e scripts para carregamento de componentes.

Publicação

  • Pronto para hospedagem em qualquer servidor estático (HostGator, UOL, Locaweb, GitHub Pages, etc).
  • Não requer backend, banco de dados ou PHP.
  • Todos os caminhos são relativos, facilitando deploy em subpastas.

Outros

  • Fontes do Google Fonts (Inter).
  • Ícones do Font Awesome CDN.
  • Compatível com SEO básico (meta tags, títulos, descrição).
